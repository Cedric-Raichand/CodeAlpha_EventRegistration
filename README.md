# 🎟️ Event Registration System (CodeAlpha Internship Project)

A backend Event Registration System built using **Node.js, Express, and MongoDB**.  
This project allows users to create events and register for them using REST APIs with proper database relationships.

---

## 🚀 Features

- Create events
- View all events
- View single event details
- Register users for events
- View all registrations
- View registrations per event
- MongoDB relational linking using ObjectId

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- dotenv
